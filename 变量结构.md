### content_list

通过以下典型例子可以清楚地了解不同`type`的`content_list`包含哪些字段

1. 纯文本类型 (type: "text")

```
"type" # 类型
"text" # 文本内容
"page_idx" # 文本所在的页码
```

2. 图片类型

```
"type" # 类型
"img_path"  # 图片路径，必须是绝对路径
"image_caption" # 图片的标题
"image_footnote" # 图片的脚注或补充说明
"page_idx" # 图片所在的页码
```

3. 表格类型

```
"type" # 类型
"table_body" # 表格内容，markdown格式
"table_caption" # 表格的标题
"table_footnote" # 表格的脚注或者补充说明
"page_idx" # 表格所在的页码
```

4. 公式类型

```
"type" # 类型
"latex" # 数学表达式。使用 LaTeX 语法存储
"text" # 对该公式的文字解释或描述，帮助 LLM 理解公式含义
"page_idx" # 公式所在的页码
```
